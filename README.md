# 🧪 Automação de Testes com Playwright - SauceDemo

## 📌 Descrição do Projeto

Este projeto tem como objetivo a automação de testes end-to-end utilizando o Playwright no site SauceDemo.

Foram desenvolvidos dois cenários de teste:

- Script 1: Fluxo de compra gerado com Playwright CodeGen 
- Script 2: Fluxo de compra implementado manualmente com boas práticas

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Playwright

---

## 🧪 Cenários Automatizados

### ✔️ 1. Fluxo com CodeGen
- Login na aplicação
- Adição de produto ao carrinho
- Validação do produto no carrinho

### ✔️ 2. Fluxo Manual
- Login na aplicação
- Adição de um produto diferente do cenário anterior
- Validação do produto no carrinho
